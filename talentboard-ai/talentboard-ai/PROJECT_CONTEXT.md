# TalentBoard AI

## Product Overview
TalentBoard AI is an AI-powered Career Operating System designed to help job seekers organize, optimize, and accelerate their job search journey.

## Core Modules
- Authentication & User Management
- Dashboard
- Job Application Tracker
- Kanban Board
- Portfolio Management
- Public Portfolio Website
- AI Career Assistant
- Analytics & Reporting
- Notifications & Reminders

## Tech Stack
- Next.js 15
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
- PostgreSQL
- OpenAI API
- Vercel

## MVP Features
- Authentication
- Dashboard
- Applications CRUD
- Kanban Board
- Portfolio Library
- Public Portfolio
- Resume Optimizer
- Interview Generator
- Portfolio Summary Generator
