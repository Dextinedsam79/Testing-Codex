# TalentBoard AI Database Schema

## Core Tables

### profiles
Stores user profile information.

### skills
Master skill catalog.

### user_skills
Maps users to skills.

### applications
Stores job applications.

### application_status_history
Tracks application stage movement.

### cv_versions
Stores resume versions.

### application_cv
Maps resumes to applications.

### projects
Stores portfolio projects.

### project_images
Stores project screenshots.

### project_skills
Maps skills to projects.

### ai_generations
Stores AI outputs.

### analytics_snapshots
Stores reporting metrics.

### notifications
Stores user notifications.

### reminders
Stores follow-up reminders.

### portfolio_settings
Stores public portfolio settings.

## Storage Buckets
- avatars
- resumes
- project-images
- portfolio-assets
